# Cheat Sheet for Git

Remember to `git status`

# Remotes

A remote is a server that is outside your computer, and stores a copy of your
repository.

Common commands - 
```
git status
git log
git log -p
git add
git add -a
git commit
git commit -m
git commit -ma
git clone
git revert
git reset HEAD 
git pull
git fetch
git push
```
